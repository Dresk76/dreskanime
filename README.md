# Dresk Anime – Página Web

## Descripción

**Dresk Anime** es una página web desarrollada como proyecto universitario cuyo objetivo es presentar contenido relacionado con el mundo del **anime y el manga**.

La página funciona como una **plataforma informativa y de redirección**, donde los usuarios pueden encontrar enlaces hacia páginas externas con contenido de anime y manga de alta calidad, evitando la necesidad de buscar en múltiples sitios.

El proyecto también fue diseñado para practicar **maquetación web, diseño visual y estructura de páginas con HTML y CSS**.

---

## Características

La página incluye varias secciones principales:

- **Inicio** con un encabezado visual y presentación del sitio
- **Acerca de** donde se explica el propósito de la página
- **Portafolio** con una galería de imágenes
- **Opiniones de usuarios** simuladas
- **Sección de servicios/productos**
- **Información de contacto en el footer**

El diseño busca ofrecer una experiencia visual atractiva con imágenes, efectos de hover y una estructura clara.

## Objetivo del proyecto

El objetivo de este proyecto fue practicar conceptos fundamentales de **desarrollo web frontend**, incluyendo:

- Estructura de páginas web con **HTML**
- Diseño y estilos con **CSS**
- Organización de contenido en secciones
- Uso de imágenes y elementos visuales
- Creación de interfaces web básicas
